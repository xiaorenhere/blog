    离忧生活 app

    # 离忧生活 app

    完成时间：2024.05-2024.07

    项目简介：对大平台的热搜榜爬取，显示实时的热搜新闻，显示添加地区天气情况及 5 天内的预测，记录当天饮食、运动情况及消耗的能量，显示自己的账号信息。

    技术栈：ArkTS +ArkUI+ Stage 模型+第三方库+用户首选项+多端部署

    # 主要内容

     数据持久化存储，使用用户首选项存储用户初始设置的少量数据，用关系型数据库存储每天饮食、运动情况及消耗的能量的数据。

     使用第三方库，用 ohpm 命令下载 axios 包，对 axios 进行二次封装，用 axios 获取实时新闻的相关数据。

     使用 http+Promise 调用申请的高德地图 api 获取相关地区天气信息，使用 Promise.all 方法，同时返回多条地区的天气数据。

     使用 ArkUI 的多种布局手段搭配原生组件搭建页面，使用 LazyForEach 渲染技术实现新闻数据懒加载。

     使用 Refresh 组件实现新闻、天气模块的下拉刷新功能，使用触底事件实现新闻长列表的上拉加载功能。

     使用媒体查询解决不同设备间的适配问题，实现一套代码工程，一次开发上架，多端按需部署。

     使用 UIAbility 组件生命周期，实现对用户首选项数据的加载，使用页面生命周期实现天气、新闻数据获取，使用组件生命周期配合@Prop、@Link 父子通信装饰器，实现天气地址、饮食运动数据的添加。

     使用@Builder、@Styles、@Extend 等装饰器封装组件或样式实现代码复用，使代码结构更加清晰。
